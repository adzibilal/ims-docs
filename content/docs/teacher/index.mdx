# Memulai: Dashboard Guru

Selamat datang di **IMS Dashboard**! Aplikasi ini dirancang untuk memudahkan Anda mengelola jadwal mengajar, mencatat absensi siswa, memantau progres materi, hingga melihat honor mengajar dengan transparan.

## Langkah 1: Cara Masuk (Login)

Untuk mulai menggunakan aplikasi, ikuti langkah mudah berikut:

1.  Buka link IMS Dashboard yang diberikan oleh Admin.
2.  Masukkan **Alamat Email** Anda.
3.  Masukkan **Password** Anda.
    *   *Tips: Jika Anda lupa password, silakan hubungi Admin untuk melakukan reset.*
4.  Klik tombol **Sign In**.

![Halaman Login Guru](/docs/teacher/login.png)

## Langkah 2: Mengenal Menu Navigasi

Di sisi kiri (sidebar), Anda akan menemukan menu-menu utama:

*   **Dashboard**: Halaman utama untuk melihat jadwal hari ini dan ringkasan cepat.
*   **My Classes**: Daftar semua kelas yang Anda pegang. Di sini Anda bisa melihat detail siswa dan riwayat pertemuan mereka.
*   **Payroll**: Halaman untuk melihat rincian honor mengajar dan mengunduh slip gaji PDF.
*   **Calendar**: Tampilan kalender untuk melihat jadwal mengajar secara visual.

![Tampilan Dashboard Guru](/docs/teacher/dashboard.png)

## Pengaturan Profil & Password

Anda dapat mengubah data diri atau mengganti password secara mandiri:

1.  Lihat ke pojok kanan atas layar Anda, klik pada **Nama/Foto Anda**.
2.  Pilih menu **My Profile**.
3.  Di halaman profil, Anda bisa:
    *   **Ubah Foto**: Klik ikon kamera pada foto profil untuk mengunggah foto baru.
    *   **Ganti Password**: Masukkan password saat ini, lalu masukkan password baru Anda, dan klik simpan.

![Pengaturan Profil](/docs/teacher/profile-settings.png)

### Tips Penting:
> [!IMPORTANT]
> **Pastikan Status Sesi Menjadi "Completed"**: Hanya sesi dengan status *Completed* (selesai) yang akan dihitung sebagai honor mengajar di akhir bulan. Status ini akan berubah otomatis jika Anda sudah mengisi absensi untuk **semua** siswa di sesi tersebut.
